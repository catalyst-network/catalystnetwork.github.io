How we do it

Any web 2.0 service is made up of a mix of data and services combined to create a rich and engaging user experience.

Web 3.0 services likewise will need to support the same level of sophistication along with broad datasets while enabling businesses to reuse and transition their data and codebases to the web 3.0 world.

Catalyst has been designed and developed from the ground up with these requirements at its heart, to create a practical environment for the rapid migration and deployment of web 3.0 services at scale.

Work in familiar languages and frameworks such as .NET, combine files and ledger datasets, work efficiently with minimal overhead.

Features

KVM

EVM compatible runtime, extended to support file actions, calls to .NET assemblies and other extended features.

Compute

Call binary processes such as .NET assemblies for complex or long running activities outside of the KVM.

Data

Work with the native distributed ledger and the native distributed file system. Generate reports, read or file files for web 3.0.

Efficient

Efficient consensus mechanism and minimal block history requirements to make Catalyst lean and green.

Modular

Use interfaces throughout to allow the rapid switching of components such as consensus, peer to peer network and mempool.

Community

Open source, following standards, .NET Foundation application, making transaction fees minimal, fair incentives and rewards.

DAY ZERO NETWORK

Users have been disappointed in past attempts at decentralisation because the networks have failed to reach viability. Catalyst breaks that mould by offering a complete, functional Day Zero Network. By using virtualised nodes as a service in conjunction with nodes run by the community, Catalyst will go live with a sustainable network in place.

REAL UTILITY

Buying into Catalyst is not buying into a promise: it’s securing a piece of a new reality. From Day Zero, there will be a viable network with functioning dApps.

ENGINEERED FOR THE COMMUNITY

The Catalyst Network is being developed by a seasoned team of tech and business veterans to provide the backbone of a fast and commercially viable decentralised network, owned and run by a truly independent community.
